# 🍽️ The World of Flavors

A 2D web-based RPG game where players explore a fantasy world inspired by food and cooking. Players can travel across different regions, interact with NPCs, complete quests, and discover unique recipes and ingredients.

## 🎮 Features

- Google Login with Firebase Authentication
- Interactive World Map
- Explore Flavoria City
- Pixel Art Graphics
- Character Movement
- NPC Interaction
- Multiple Maps
- Future Quest System
- Inventory System
- Battle System (Coming Soon)

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Firebase Authentication
- Firebase Hosting
- Canvas API

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/TheWorldOfFlavors.git
```

### Open the project

Using Python

```bash
python -m http.server 5500
```

or use the VS Code **Live Server** extension.

Then open:

```
http://localhost:5500
```
